### Hotel Oxygen - Online Booking Website



Hotel Oxygen is a responsive hotel reservation website built with HTML, CSS, JavaScript. The project showcases hotel rooms, facilities, menu offerings, news/blog updates, and a basic booking form for checking room availability.

#### 

##### Features

* Responsive navigation bar with menu toggle
* Hero section with call-to-action
* Booking form for arrival date, departure date, and number of guests
* About section highlighting hotel services and values
* Room listings with descriptions and prices
* Intro video section
* Facilities and core features display
* Food menu showcase
* News/blog section
* Footer with contact details and social media links
* Scroll animations using ScrollReveal.js
* Icons powered by **Remix Icon**





##### Technologies Used

* HMTL 5
* CSS3
* JavaScript(Vanilla JS)
* ScrollReveal.js
* Remix Icon CDN

##### How to Run the Project

1. Download or clone the repository
2. Ensure all files (index.html, styles.css, main.js, and assets folder) are in the same directory
3. Open index.html in your web browser

#### 

##### Notes

* The booking form is currently frontend-only and does not submit data to a backend
* Dates and guest inputs are placeholders and can be enhanced with validation or a booking API
* The project is suitable for learning, practice, and portfolio showcase
